# GitGrade Analyzer

GitGrade is an AI-inspired system that evaluates a GitHub repository
and generates a score, summary, recruiter-style verdict, and a
personalized improvement roadmap for students.

## Problem Statement
Convert a GitHub repository into a meaningful evaluation that reflects
its real strengths and weaknesses.
